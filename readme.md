## Tutor for bot home
	* git clone https://github.com/ikiganteng/toolsigeh.git
	* cd toolsigeh
	* php home.php
	* masukkan username, password
  	* nanti akun lu akan tersimpan di instagram.txt
	* edit file bothome.php sesuai akun lu
  	* run php bothome.php
	* enjoy it!
